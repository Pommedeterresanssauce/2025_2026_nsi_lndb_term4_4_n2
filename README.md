# Poker 2, le poker mais en mieux.

Animations :
   - Cartes qui bougent quand la souris passe dessus
   - Cartes qui tournent lorsqu'elles sont sélectionnées

Gameplay :
   - Ce qu'on garde du poker :
      - Système de mise de jetons
      - Combinaisons
   - Ce qu'on ajoute :
      - 2 cartes en plus dans la main
      - Jokers inclus
      - Nouvelles cartes (à étudier)
      - Items :
         - Bière
         - Loupe
         - Flingue
         - Allumette
         - Biscuit
         - Gilet pare-balle
      - Nouvelles combinaisons
      - Nouvelles cartes
   
Règles :
   - 1er tour :
      - Pioche initiale des mains (2 cartes)
      - Pioche initiale des objets
      - Mise initiale
      - Flop (3 premières cartes retournées)
      - Pioche des 2 cartes restantes
      - Autorisation de jouer les objets
      - Partie normale jusqu'à la dernière mise
      - Sélection de 2 cartes à jouer dans sa main
      - Fin du tour et désignation du gagnant
   - Autres tours :
      - Pas de pioche d'objet pour le gagnant et ceux ayant déjà 2 objets

Organisation du code :
   - main.py :
      - Programme de lancements
